What's included
src/ingest.py — The one-time indexing pipeline:

Loads PDF (via pdfplumber), TXT, and DOCX files from /data
Splits into 500-word chunks with 80-word overlap
Batch-embeds all chunks via OpenAI text-embedding-3-small
Saves a FAISS cosine-similarity index + metadata to /index/

src/query.py — The interactive CLI loop:

Loads the persisted index
Embeds each user question, runs top-K similarity search
Passes retrieved chunks to gpt-4o-mini with a strict system prompt that forbids answering outside the context
Displays the answer + source chunk previews with similarity scores

src/app.py — Optional Streamlit web UI with a dark, polished design (bonus marks territory)
README.md — Fully covers all 9 required sections: architecture diagram, chunking rationale, model choices, setup steps, environment variables, 6 example queries, and known limitations.
